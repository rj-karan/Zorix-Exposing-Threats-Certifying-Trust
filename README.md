# Zorix — Exposing Threats, Certifying Trust

AI-powered vulnerability detection platform with a FastAPI backend and React dashboard frontend.

# 🚀 Project Overview

Zorix analyzes GitHub repositories to detect vulnerabilities using AI-driven reasoning and structured analysis pipelines.

The platform includes:

- FastAPI Backend (AI + Database + Auth)
- React Dashboard Frontend
- GitHub Code Analysis
- AI Vulnerability Detection
- Docker Deployment Support
